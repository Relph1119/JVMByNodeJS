# 记录自己用NodeJS完成编写JVM的过程

项目完全参考张秀宏的《自己动手写Java虚拟机》代码结构，在此向本书作者表示感谢。

## 运行环境
NodeJS 版本：12.11.1  
WebStorm 版本：WebStorm 2018.2.3  
Java版本：1.8
